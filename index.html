Color

Color is used to draw attention to the important sections of the website so as not to overwhelm the user. The primary color is used to draw attention as it is the color the strongest contrast. The secondary color is used in occasions where there still needs to be a contrast but the primary and tertiary colors are already in use. The tertiary color is used predominantly for text color. The background is for the background of the webpage. On occasion when there is need, it is used for a font color against the tertiary color.
